Prism.js — Демонстрация подсветки кода

Эта страница демонстрирует возможности библиотеки Prism.js — подсветку синтаксиса, переключение тем, копирование кода и управление размером шрифта. Ниже — примеры кода на Python, HTML, CSS и JavaScript.

Вы можете переключать темы, чтобы посмотреть, как они влияют на читаемость кода и обычного текста.

1. Пример на Python: Рекурсия

Функция для вычисления факториала с помощью рекурсии.


def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

# Вызов
result = factorial(5)
print(f"Факториал 5 = {result}")

Рекурсия — мощный инструмент, но требует осторожности: при большом числе вызовов возможен переполнение стека.

4. Пример на JavaScript

Функция для поиска максимального числа в массиве.


function findMax(arr) {
  if (arr.length === 0) return null;
  return Math.max(...arr);
}

const numbers = [3, 7, 2, 9, 1];
console.log("Максимум:", findMax(numbers));

Функция использует spread-оператор ... и встроенную функцию Math.max().

Prism.js автоматически подсвечивает синтаксис всех поддерживаемых языков: Python, HTML, CSS, JavaScript, Bash и многие другие.

5. Команда Windows (cmd)

Поиск всех текстовых файлов в папке пользователя.


dir /s /b C:\Users\*.txt

Эта команда полезна для поиска файлов по расширению.
