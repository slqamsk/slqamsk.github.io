Что такое HTML?

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание веб-страницы с помощью системы тегов и атрибутов.

Основные элементы HTML

Структура документа

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Заголовок страницы</title>
</head>
<body>
  <h1>Мой первый заголовок</h1>
  <p>Мой первый абзац.</p>
</body>
</html>

Текстовые элементы

<h1>-<h6> - Заголовки разных уровней
<p> - Абзац текста
<a href="url"> - Ссылка
<strong> - Важный текст
<em> - Акцентированный текст

Списки и таблицы

Нумерованный список

<ol>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
</ol>

Таблица

<table>
  <tr>
    <th>Заголовок 1</th>
    <th>Заголовок 2</th>
  </tr>
  <tr>
    <td>Данные 1</td>
    <td>Данные 2</td>
  </tr>
</table>
