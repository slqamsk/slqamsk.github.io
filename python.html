Основные операторы в Python

Python - это мощный и удобный язык программирования с простым синтаксисом. В этой статье рассмотрим основные операторы Python с примерами кода.

1. Арифметические операторы

Python поддерживает все основные арифметические операции:

# Сложение, вычитание, умножение, деление
print(10 + 5) # 15
print(10 - 5) # 5
print(10 * 5) # 50
print(10 / 5) # 2.0

# Целочисленное деление и остаток от деления
print(10 // 3) # 3
print(10 % 3) # 1

# Возведение в степень
print(2 ** 3) # 8

2. Операторы сравнения

Используются для сравнения двух значений и возвращают True или False:

print(10 == 10) # True
print(10 != 5) # True
print(10 > 5) # True
print(10 < 5) # False
print(10 >= 10) # True
print(10 <= 5) # False

3. Логические операторы

Используются для комбинирования условных операторов:

# and - логическое И
print(True and False) # False

# or - логическое ИЛИ
print(True or False) # True

# not - логическое НЕ
print(not True) # False

Пример использования в условии

if age >= 18 and age <= 65:
print("Вы можете работать")

4. Операторы присваивания

Используются для присваивания значений переменным:

# Простое присваивание
x = 10

# Комбинированные операторы
x += 5 # x = x + 5
x -= 3 # x = x - 3
x *= 2 # x = x * 2
x /= 4 # x = x / 4

5. Операторы принадлежности

Проверяют, есть ли значение в последовательности:

numbers = [1, 2, 3, 4, 5]
print(3 in numbers) # True
print(6 not in numbers) # True

6. Операторы идентичности

Проверяют, ссылаются ли переменные на один и тот же объект:

x = [1, 2, 3]
y = x
z = [1, 2, 3]

print(x is y) # True (это один и тот же объект)
print(x is z) # False (разные объекты с одинаковым содержимым)
print(x is not z) # True

7. Установка Python и запуск скриптов

Для работы с Python его нужно сначала установить. Вот как это сделать в разных ОС:

Установка на Windows

Рекомендуется использовать официальный установщик с сайта python.org

После установки проверьте версию Python:

python --version

Запуск скриптов

Чтобы выполнить Python-скрипт, используйте команду:

python script.py

Где script.py - имя вашего файла с кодом.

8. Пример программы

Вот пример простой программы на Python, использующей несколько операторов:

# Программа проверяет, является ли число простым
def is_prime(n):
if n <= 1:
return False
for i in range(2, int(n**0.5) + 1):
if n % i == 0:
return False
return True

# Проверка работы функции
number = int(input("Введите число: "))
if is_prime(number):
print(f"Число {number} является простым")
else:
print(f"Число {number} не является простым")
# Пример вывода программы:
Введите число: 17
Число 17 является простым
