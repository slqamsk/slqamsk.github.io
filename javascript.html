Что такое JavaScript?

JavaScript - это язык программирования, который позволяет создавать динамически обновляемый контент, управлять мультимедиа, анимировать изображения и многое другое. Это одна из основных технологий веба наряду с HTML и CSS.

Основы синтаксиса

Переменные и типы данных

// Объявление переменных
let message = "Привет, мир!"; // Строка
const PI = 3.14;              // Число, константа
var isActive = true;          // Булево значение

// Объекты
const user = {
  name: "Алексей",
  age: 30,
  isAdmin: false
};

// Массивы
const colors = ["красный", "зелёный", "синий"];

Функции

// Объявление функции
function greet(name) {
  return `Привет, ${name}!`;
}

// Стрелочная функция
const sum = (a, b) => a + b;

// Вызов функций
greet("Мир"); // "Привет, Мир!"
sum(2, 3);    // 5

Работа с DOM

DOM (Document Object Model) - это программный интерфейс для HTML и XML документов, который представляет страницу в виде дерева объектов.

Примеры работы с DOM

// Получение элементов
const header = document.querySelector('header');
const buttons = document.querySelectorAll('button');

// Изменение контента
document.getElementById('title').textContent = 'Новый заголовок';

// Обработчики событий
document.getElementById('myButton').addEventListener('click', () => {
  alert('Кнопка нажата!');
});

// Создание элементов
const newDiv = document.createElement('div');
newDiv.className = 'alert';
newDiv.textContent = 'Важное сообщение';
document.body.appendChild(newDiv);
