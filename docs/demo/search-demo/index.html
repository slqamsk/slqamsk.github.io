Поиск элементов на веб-странице

Элементы (теги) на веб-странице можно найти с помощью 6 специфичных методов:

  • По атрибуту id
  • По атрибуту name
  • По атрибуту class
  • По имени тега
  • По полному тексту ссылки
  • По частичному тексту ссылки

Формы для поиска по атрибутам

Эти элементы форм можно найти по атрибутам id, name и class:

(id="username", name="user_name", class="form-input")
(id="user-email", name="user_email", class="form-input email-field")
(id="user-password", name="user_password", class="form-input password-field")
(name="interests", class="interest-checkbox")
(id="country", name="user_country", class="form-select")
