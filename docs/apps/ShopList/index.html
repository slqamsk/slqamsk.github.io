ShopList — Бизнес-требования (BRD)

BRD — ShopList (минимальный функционал)

Назначение:
Веб-приложение для ведения семейного списка покупок. Позволяет просматривать продукты,
добавлять их в список покупок, указывать количество, отмечать статус ("Купил", "Не было"),
управлять списком и удалять записи.

Страницы:

Главная страница:
- Отображает краткое описание системы: это приложение для ведения семейного списка покупок.

Общий список продуктов:
- Показывает все продукты.
- Переключатель: "Все продукты" / "Не в списке покупок".
- У каждого продукта — кнопка "Добавить в список покупок".
- Кнопка "Добавить новый продукт" ведёт на страницу создания.

Список покупок:
- Отображает все добавленные продукты.
- Для каждого: название, поле для количества (текст), кнопки "Купил" и "Не было".
- Продукты со статусом "Купил" или "Не было" остаются в списке, но визуально выделяются.
- Две кнопки управления: "Удалить всё" и "Удалить купленные".

Добавление/редактирование продукта:
- Поле ввода названия (обязательное, 2–80 символов, не только пробелы).
- Кнопки: "Сохранить", "Отмена", в режиме редактирования — дополнительно "Удалить".

Общее поведение:
- На всех страницах — общее меню: Главная | Общий список | Список покупок | Добавить продукт.
- Все действия сопровождаются уведомлениями (успех или ошибка), которые остаются на экране.
- При выполнении операций показывается индикатор загрузки, интерфейс блокируется.
- После удаления всего списка или купленных — страница обновляется.
- Интерфейс обновляется без перезагрузки, кроме ключевых операций (очистка списка и т.п.).

Коротко:
Веб-приложение для семейного списка покупок: можно добавлять продукты, формировать список,
отмечать статус и управлять содержимым.
