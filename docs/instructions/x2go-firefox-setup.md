# ИНСТРУКЦИЯ: УСТАНОВКА X2Go КЛИЕНТА НА WINDOWS И НАСТРОЙКА ПОДКЛЮЧЕНИЯ ТОЛЬКО БРАУЗЕРА

## ПРЕДВАРИТЕЛЬНЫЕ УСЛОВИЯ (ЧТО ДОЛЖНО БЫТЬ НА СЕРВЕРЕ И КАК ПРОВЕРИТЬ)

На сервере Linux (Ubuntu) уже должны быть установлены: X2Go Server, Firefox, а также окружение Xfce. Для проверки выполните через SSH:

**Проверка X2Go Server:**
```bash
systemctl status x2goserver
```
Если не установлен:
```bash
sudo apt install x2goserver x2goserver-xsession
```

**Проверка Firefox:**
```bash
which firefox
```
Если не установлен:
```bash
sudo apt install firefox
```

**Проверка Xfce:**
```bash
dpkg -l | grep xfce4-session
```
Если не установлен:
```bash
sudo apt install xfce4 xfce4-goodies
```

**Проверка SSH:**
```bash
systemctl status ssh
```

## УСТАНОВКА КЛИЕНТА X2Go НА WINDOWS

1. Откройте страницу загрузки X2Go:  
   `https://wiki.x2go.org/doku.php/download:start`

2. Найдите раздел **"X2Go Client"**. В строке "Download" нажмите на ссылку **"mswin"**.

3. Или скачайте по прямой ссылке:  
   `http://code.x2go.org/releases/X2GoClient_latest_mswin32-setup.exe`

4. Запустите скачанный `.exe` файл.

5. Следуйте инструкциям мастера установки:  
   - примите лицензионное соглашение  
   - выберите папку установки (оставьте по умолчанию)  
   - компоненты (оставьте все по умолчанию)

6. Завершите установку. Ярлык **X2Go Client** появится в меню "Пуск".

## НАСТРОЙКА ПОДКЛЮЧЕНИЯ ДЛЯ ЗАПУСКА ТОЛЬКО БРАУЗЕРА

1. Запустите **X2Go Client**.

2. Нажмите **"Session" → "New Session"** (или **"Новая сессия"**).

3. Заполните параметры на вкладке **"Session"**:

   | Поле | Значение |
   |------|----------|
   | Session name | любое имя, например `Firefox на сервере` |
   | Host | `XXX.XXX.XXX.XXX` (IP вашего сервера) |
   | Login | ваше_имя_пользователя |
   | SSH port | `22` |
   | Session type | `Single Application` (или `Одно приложение`) |
   | Command | `firefox` |

4. Перейдите на вкладку **"Media"** (или **"Медиа"**):

   - Снимите галочку **"Client-side printing"** (Печать на стороне клиента)
   - **Sound**: выберите `none` (отключить звук)

5. Нажмите **"OK"** для сохранения сессии.

---

## ПОДКЛЮЧЕНИЕ К СЕРВЕРУ

1. Дважды кликните по созданной сессии или выберите её и нажмите **"Connect"**.

2. При первом подключении появится предупреждение о проверке ключа хоста. Нажмите **"Да" (Yes)**.

3. Введите пароль вашего пользователя на сервере и нажмите **"OK"**.

4. Через несколько секунд откроется окно **Firefox**, работающего на сервере.

## ПРОВЕРКА, ЧТО БРАУЗЕР НА СЕРВЕРЕ

### Способ 1: Проверка IP адреса
Откройте в Firefox сайт для проверки IP адреса: **2ip.ru**  
Он должен показать IP адрес вашего сервера (тот же, который вы ввели в Host), а не IP вашего домашнего Windows.

### Способ 2: Проверка процессов
В адресной строке Firefox введите: `about:processes`  
В открывшемся менеджере процессов найдите строку **"Fork Server"**.  
Это компонент, который существует только в Linux версии Firefox. В Windows версии такого процесса нет. Если вы видите **"Fork Server"** — значит Firefox точно работает на сервере Linux.
