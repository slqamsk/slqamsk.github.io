# 📚 Инструкции
В этом разделе собраны пошаговые руководства по настройке различных инструментов и сервисов.

1. [Настройка GitHub Pages для отображения Mermaid-диаграмм](./mermaid-github-pages-setup.html)
Как настроить репозиторий на GitHub Pages так, чтобы Mermaid-диаграммы в `.md` файлах отображались графически, а не как исходный код.

2. [Промпт для преобразования Markdown в HTML](./prompt-guide.html)
Как с помощью специального промпта заставить ИИ оформлять ответы в виде готовых HTML-страниц с Markdown-разметкой.

3. [Настройка X2Go клиента на Windows для подключения к Firefox на сервере](./x2go-firefox-setup.txt)
Пошаговая инструкция по установке X2Go Client на Windows, настройке подключения к серверу Linux и запуску только браузера (режим Single Application) с проверкой через IP и менеджер процессов.

4. [Настройка нового проекта для автотестирования в IntelliJ IDEA](./at-java-web-new-project.md)
   Пошаговая инструкция по настройке `build.gradle` для автоматического тестирования с использованием Selenide, включая подключение зависимостей, настройку логирования и поддержку кириллицы.
