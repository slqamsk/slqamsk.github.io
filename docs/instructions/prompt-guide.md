# Как получить HTML-страницу с Markdown от ИИ

Используй этот промпт после того, как ИИ дал текстовый ответ, который нужно оформить.

---

## Промпт

```
Отформатируй предыдущий ответ как Markdown и вставь в этот HTML:

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>ЗАГОЛОВОК_ТЕМЫ</title>
    <script type="module" src="https://cdn.jsdelivr.net/npm/zero-md@3?register"></script>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/prismjs@1/themes/prism.css">
</head>
<body>
    <zero-md>
        <script type="text/markdown">
СЮДА_ОТВЕТ_В_MARKDOWN
        </script>
    </zero-md>
</body>
</html>

Важно:
- Замени ЗАГОЛОВОК_ТЕМЫ на название статьи.
- Для приблизительных дат используй ≈ вместо ~ (тильда вызывает ~~зачёркивание~~).
- Для переноса строк внутри одного блока используй два пробела в конце строки.
- Не добавляй своих CSS-стилей — шаблон уже готов.
- Верни только HTML, без пояснений.
```

---

## Как использовать

Скопируй этот промпт и вставь в чат с ИИ **сразу после того, как получил от него текстовый ответ**, который хочешь превратить в HTML-страницу.

ИИ возьмёт свой предыдущий ответ, отформатирует его как Markdown, вставит в шаблон и вернёт готовый `.html` файл. Тебе останется только сохранить его и открыть в браузере или загрузить на сайт.
