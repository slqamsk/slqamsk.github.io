Система контроля температуры

Регулятор температуры лабораторного помещения
Для безопасности оборудования температура изменяется плавно.
Резкие скачки (более 10°C) могут вывести систему из строя.
Прямой ввод температуры отключен в целях безопасности.
Выберите приблизительное значение, затем используйте кнопки +/- для точной настройки.
