Требования к форме расчёта стоимости билета в кино

Поля формы
FR-01 Поле возраст (число полных лет) - целое число от 0 до 99
FR-02 Дата сеанса - 7 дней подряд начиная с завтрашнего числа (на сегодня билет купить уже нельзя)
FR-03 Начало сеанса - возможные значения: 09:30, 11:00, 12:30, 14:00, 16:00, 17:30, 19:00, 20:30, 22:00, 23:30
FR-04 Фильм - возможные значения:
    • Назад в будущее - PG (12+)
    • Криминальное чтиво - R (18+)
    • Король-лев - G (0+)
    • Прирождённые убийцы - NC-17 (18+)
    • Последнее танго в Париже - NC-17 (18+)

Валидация заполнения полей
FR-05 Если какие-то из полей не заполнены, то при нажатии кнопки "Рассчитать" вместо стоимости билета выводится сообщение о том, какие поля не заполнены

Проверка возрастных ограничений
FR-06 Фильмы с маркировкой "G (0+)" разрешены для любого возраста
FR-07 Фильмы с маркировкой "PG (12+)" разрешены только с 12 лет
FR-08 Фильмы с маркировкой "R (18+)" и "NC-17 (18+)" разрешены только с 18 лет
FR-09 Если указан возраст, недостаточный для просмотра фильма, то при нажатии кнопки "Рассчитать" вместо стоимости билета выводится сообщение о том, что возраст не подходящий

Расчет стоимости билета
FR-10 Базовая цена билета составляет 500 рублей
FR-11 Предусмотрены следующие скидки:
FR-11.1 Скидка для детей: детям, которым ещё не исполнилось 6 лет скидка 250 рублей
FR-11.2 Скидка буднего дня: по будним дням скидка 100 рублей
FR-11.3 Скидка на утренние сеансы: на сеансы до 14:00 скидка 50 рублей
FR-12 Все скидки суммируются
