Функциональные требования для системы вентиляции двух лабораторий

Есть две лаборатории и одна система вентиляции для них.
Вентиляция может либо не работать, либо использоваться одной из лабораторий.
Вентиляция не может использовать одновременно в обеих лабораториях.

FR-01. Система должна поддерживать глобальный статус использования вентиляции:
	"Не используется"
	"Используется 1-й лаборатория"
	"Используется 2-й лаборатория"
FR-02. Система должна отображать две панели управления: "Лаборатория-1" и "Лаборатория-2".
FR-03. Каждая панель должна содержать:
	Заголовок с названием лаборатории.
	Индикатор локального статуса вентиляции. Возможные значения: "Свободно", "Работает", "Занято".
	Три кнопки: "Обновить", "Включить", "Выключить".
	Локальный статус вентиляции в одной лаборатории не должен изменяться автоматически при включении/выключении вентиляции в другой лаборатории.
FR-04. Кнопки должны быть доступны или недоступны в зависимости от локального статуса вентиляции:
	Если статус "Свободно": "Обновить" — доступна, "Включить" - доступна, "Выключить" — недоступна.
	Если статус "Занято": "Обновить" — доступна, "Включить" - недоступна, "Выключить" — недоступна.
	Если статус "Работает": "Обновить" — недоступна, "Включить" - недоступна, "Выключить" — доступна.
FR-05. При открытии страницы:
	Глобальный статус вентиляции должен быть "Свободно".
	Локальные статусы вентиляций в обеих лабораториях должны быть "Свободно".	
FR-06. При нажатии кнопки "Обновить":
	Локальный статус вентиляции обновляется в соответствии с глобальным статусом вентиляции.
FR-07. При нажатии кнопки "Включить":
	Глобальный статус вентиляции должен измениться на "Работает X-я лаборатория", где X — номер лаборатории.
	Локальный статус вентиляции в этой лаборатории должен измениться на "Работает".
FR-08. При нажатии кнопки "Выключить":
	Глобальный статус вентиляции должен измениться на "Свободно".
	Локальный статус вентиляции в этой лаборатории должен измениться на "Свободно".
